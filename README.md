# PYCLIGram
## A simple Telegram CLI client written in Python
# Getting started
## You will need:
- A Telegram account
- Python 3.8 or higher installed on your computer
- App `api_id` and App `api_hash` from [my.telegram.org/apps](https://my.telegram.org/apps)
- Telethon and Colorama libraries (`pip install telethon && pip install Colorama`)
### Step 1: Download or clone the repository
### Step 2: Install dependencies(pip install telethon)
### Step 3: Open `PYCLIGram.py` in your preferred editor
### Step 4: Get your `api_id` and `api_hash` from [my.telegram.org/apps](https://my.telegram.org/apps)
### Step 5: Put your `api_id` and `api_hash` into the fields in the code
### Step 6: Run the code(PYCLIGram.py)
### Step 7: Choose a mode and sign in
