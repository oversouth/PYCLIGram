# PYCLIGram
## A simple Telegram CLI client written in Python
# Getting started
## You will need:
- A Telegram account
- Python 3.8 or higher installed on your computer
- App `api_id` and App `api_hash` from [my.telegram.org/apps](https://my.telegram.org/apps)
- Telethon and Colorama libraries (`pip install telethon && pip install Colorama`)
### Step 1: Download or clone the repository
### Step 2: Get your `api_id` and `api_hash` from [my.telegram.org/apps](https://my.telegram.org/apps) (free!)
### Step 3: Put your `api_id` and `api_hash` into the fields which say #GET ONE YOURSELF AT https://my.telegram.org
### Step 4: Run PYCLIGram.py
### Step 5: Choose a mode and sign in
